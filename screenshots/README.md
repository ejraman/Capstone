# Screenshots of Streamlit Pages

These PNGs were generated programmatically from a sampled dataset and saved here for quick review.

Files:

- `0_Dashboard.png` — Executive dashboard snapshot
- `1_Overview.png` — Overview page (sample table)
- `2_Salary_Insights.png` — Salary Insights (sample table)
- `3_Company_Trends.png` — Company Trends (sample table)
- `4_Skills_Analysis.png` — Skills Analysis (sample table)

You can regenerate the images by running:

```
python scripts/generate_screenshots.py
```

Adjust `scripts/generate_screenshots.py` to change sample size or add details to the images.
